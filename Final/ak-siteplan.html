The Color Scheme

Primary Color

This will be used as the primary color on the website. It is a darker blue which can help people feel calm and trust the site. It will be used in some of the main headings and can also be found in the footer..

Secondary Color

This secondary color will be used as a filler color or a background color on some pages of the website. It is a simple off white and can be easy to read when under text and can also be found in the footer..

Accent Color #1

This accent color will be used in the header and the footer of the page and can be found in the logo of the page.

Accent Color #2

This accent color will show up when any links are hovered over and can also be found in the footer.

The Typography

Montserrat

This font will be used for the headings of the websire. It is a san serif which is simple and easy to read and see while also being professional.

Fira Sans

The font Fira Sans pairs well with Montserrat and is a good contrast. While Montserrat is thicker and heavier, Fira Sans is a thin smaller type face that will work well as captions and smaller areas of text.
